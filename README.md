# Oceania Honeymoon

一趟從台灣出發，跨越新加坡、澳洲、紐西蘭的蜜月旅行規劃。

**🌏 線上行程頁面：[ytzc.github.io/oceania-honeymoon](https://ytzc.github.io/oceania-honeymoon/)**

---

## 關於這趟旅行

這是我們第一次一起走這麼遠。
從熱帶的新加坡，到南半球的秋天，再到紐西蘭的峽灣與山脈——
不趕時間，只是想好好在一起，看看這個世界。

---

## 旅行期間

**2026 年 4 月 29 日（三）～ 5 月 17 日（日）**，共 19 天。

---

## 行程總覽

```
台灣（桃園）
  ↓ 酷航 TR867
新加坡（樟宜）— 2 晚
  ↓ 捷星 JQ008（飛機上過夜）
澳洲（墨爾本）— 2 晚
  ↓ 捷星 JQ171
紐西蘭 基督城 — 3 晚
  ↓ 自駕 / 巴士（待確認）
紐西蘭 皇后鎮 — 9 晚
  ↓ 捷星 JQ292
奧克蘭（轉機，同日）
  ↓ 中華航空 CI054（飛機上過夜）
台灣（桃園）
```

## 預算摘要

目前已確認的機票費用：

| 航段 | 航空公司 | 金額（NTD）|
|------|----------|-----------|
| 台北 → 新加坡 | 酷航 | 9,328 |
| 新加坡 → 墨爾本 | 捷星 | 21,156 |
| 墨爾本 → 基督城 | 捷星 | 15,694 |
| 皇后鎮 → 奧克蘭 → 台北 | 捷星 + 中華航空 | 33,864 |
| **機票合計** | | **80,042** |
| **每人平均** | | **~40,021** |

> 住宿、餐飲、交通、活動預算待補充。

## Repo 結構

```
oceania-honeymoon/
├── README.md                ← 專案首頁（本文件）
├── docs/
│   └── index.html           ← GitHub Pages 行程網頁
├── notes/
│   ├── plan.md              ← 主規劃文件：航班、時間軸、住宿、預算、清單
│   ├── gemini-map.md        ← 用 Gemini 生成旅遊地圖教學
│   └── ...                  ← 未來：景點筆記、照片說明等
└── .github/workflows/
    └── deploy.yml           ← 推 tag 自動部署到 GitHub Pages
```

主要規劃文件為 [`notes/plan.md`](./notes/plan.md)，包含：
- 航班總覽（含訂票者、行李額、廳別）
- 逐日行程時間軸
- 住宿待辦清單
- 已知預算
- 行前準備清單
- 重要備註與注意事項
