## ⚠️ 重要備註 Important Notes

### 行李規則

| 航段 | 託運行李 | 備註 |
|------|----------|------|
| TPE → SIN（TR867）| 雙人共 20kg | 酷航低成本，超重費用高 |
| SIN → MEL（JQ008）| 雙人共 20kg | 捷星低成本，注意重量 |
| MEL → CHC（JQ171）| 雙人共 30kg | 比去程多 10kg |
| ZQN → AKL（JQ292）| 每人 30kg | 擎訂 |
| AKL → TPE（CI054）| 每人 **2×23kg**（共 4 件）| 中華航空優惠！行李直掛（不用在 AKL 重新託運）|

> **提醒：** 去程總行李額較少（20kg / 段），回程中華航空非常寬裕（每人 2×23kg）。注意在紐西蘭採購的物品重量。

### 回程轉機資訊（5/16）

- **ZQN → AKL（JQ292）**
  - 10:50 出發，12:40 抵達奧克蘭 D 廳
  - **轉機時間約 7 小時**（等待 CI054 19:30）
  - 需從 D 廳移至 I 廳
  - 確認是否需要在奧克蘭入境 / 過境

- **AKL → TPE（CI054）**
  - 19:30 出發，途經布里斯本（BNE）停留約 90 分鐘
  - 行李直掛，**無需在布里斯本重新辦理**
  - 05:40+1 抵達桃園

### 其他注意事項

- **5/16 早上皇后鎮出發：** JQ292 10:50 出發，建議 09:00 前到達機場（皇后鎮機場較小，但蜜月季節旅客多）
- **SIN → MEL 飛機上過夜：** 5/1 21:00 出發，5/2 06:15 抵達，記得準備頸枕 / 眼罩
- **基督城機場廳別：** JQ171 抵達 I 廳（International），注意和 T1/T2 不同

## 🧠 優化建議 Suggestions

### 短期待辦
1. **住宿預訂** — 尤其是皇后鎮（5/1～5/16 旺季），建議優先訂
2. **NZeTA 申請** — 紐西蘭旅遊稅 + 電子旅行授權，需提前在 app 上申請
3. **澳洲 ETA** — 可用 app 線上申請，通常幾分鐘即核准
4. **基督城 → 皇后鎮 交通確認** — 自駕（推薦）或 InterIslander / 巴士

### 自動化 / 工具建議

| 需求 | 工具建議 |
|------|----------|
| 匯率追蹤 | [Wise](https://wise.com) 設定 TWD→NZD/AUD 匯率提醒 |
| 旅程提醒 | Google Calendar + 航班資訊 iCal 訂閱 |
| 行程地圖 | Google My Maps 或 Wanderlog 規劃路線 |
| 文件備份 | Google Drive 建立旅遊資料夾，護照 / 訂位紀錄 PDF |
