## 快速使用指南

### ✈️ 班機延誤時先做什麼

| 步驟 | 動作 |
|------|------|
| **1. 當下保留** | 登機證（原訂班次）、向航空公司服務台索取**書面延誤聲明**（需蓋章，說明原因與延誤時數）、任何因延誤產生的餐飲/住宿**收據** |
| **2. 找哪張保單** | **兆豐産險**（適用 SIN/MEL/NZ）+ **富邦旅平險**（SIN/MEL/NZ 用 3226CT20D09794，OKI 用 3226CT20D09791）|
| **3. 可申請保障** | 班機延誤津貼（每超過規定時數補一筆定額；門檻時數需查 PDF 確認） |
| **4. 特別注意** | 延誤需超過門檻才觸發（通常 4 或 6 小時）；**書面聲明須包含官方原因**；兆豐和富邦各自獨立申請，但需確認是否有「不得重複理賠」條款 |

---

### 🧳 行李不見時先做什麼

| 步驟 | 動作 |
|------|------|
| **1. 當下保留** | 下機後**立即**到行李服務台報案，取得 **PIR（Property Irregularity Report）**；保留登機證和行李標籤貼紙 |
| **2. 找哪張保單** | **兆豐産險**（SIN/MEL/NZ）+ **富邦旅平險**（OKI 或 SIN/MEL/NZ 依行程） |
| **3. 可申請保障** | **行李延誤**（超過 12~24 小時未到，可報購買緊急必需品費用）；確認遺失後申請**行李遺失賠償**（需提供清單與估值） |
| **4. 特別注意** | 手機、電腦、現金、珠寶通常**不在行李遺失保障範圍**；正式確認遺失通常需等 21 天；延誤與遺失是兩個不同申請，先申延誤再申遺失 |

---

### 🏥 海外就醫時先做什麼

| 步驟 | 動作 |
|------|------|
| **1. 當下保留** | 醫療費用收據**正本**（不得塗改）、診斷書（英文或中文）、處方箋、醫院收費明細 |
| **2. 找哪張保單** | **兆豐産險**緊急救援電話 **0800-053-588**（SIN/MEL/NZ）；富邦旅平險傷害醫療 + 海外突發疾病 |
| **3. 可申請保障** | 急診/住院醫療費用；緊急醫療協助（兆豐 0800 可協助安排就醫/翻譯） |
| **4. 特別注意** | **事先儲存 0800-053-588**（兆豐緊急救援，國際直撥需另確認號碼）；海外突發疾病**不含旅行前已知的既有疾病/慢性病**；醫療費用需保留正本，不接受影本 |

---

### 🚗 租車擦撞時先做什麼

| 步驟 | 動作 |
|------|------|
| **1. 當下保留** | 租車公司損傷確認報告、修車費明細或自負額**扣款收據**（需為官方文件）、租車契約、若有事故建議報警取得**警察事故報告** |
| **2. 找哪張保單** | NZ 用 **富邦 3226CT20D09795**（SIN/MEL/NZ 租車附加）；OKI 用 **富邦 3226CT20D09792**（OKI 租車附加）|
| **3. 可申請保障** | 租車公司向你收取的**自負額（CDW/NOC/超額扣款）**補償 |
| **4. 特別注意** | 只補你被收的「自負額」部分，**非車輛全額修繕**；NZ 租車（Snap Rentals）已含 Simply Better Cover，OKI（Dream Rental Car）已含①免責補償+②NOC，富邦為額外補充；自負額上限需查 PDF 確認 |

## 3. 我實際旅途中可能用到的保障

### ✈️ 班機延誤

- 若班機延誤超過規定時數（兆豐/富邦門檻均需查 PDF），每超過 X 小時可領一筆津貼
- 兆豐（SIN/MEL/NZ）與富邦旅平險**均可能涵蓋班機延誤**，但需確認是否可同時向兩家申請或有重疊限制
- 須向航空公司取得**書面延誤聲明（加蓋官章）**
- 天候延誤/機械故障均可申請，但轉機自願錯過不算

### 🧳 行李遺失 / 延誤

- **行李延誤**：到達目的地後超過 12~24 小時（需查確切門檻）行李未到，可申請購買緊急必需品費用
- **行李遺失**：航空公司最終確認遺失後，可申請賠償（通常需等 21 天）
- 貴重物品（手機、筆電、現金、珠寶）通常**不在賠償範圍**或有很低的單件上限

### 🏥 海外就醫（意外或突發疾病）

- **兆豐 0800-053-588**：緊急時先打，可協助安排就醫和緊急救援（NZ/AUS）
- 急診 / 住院費用：憑收據正本回國後申請（富邦傷害醫療 + 兆豐醫療）
- **不含**旅行前已知的既有疾病或慢性病急性發作（視保單條款而定）
- 沖繩就醫：僅富邦 OKI 保單（兆豐不含沖繩行程）

### 🚗 租車碰撞 / 刮傷

- **NZ**：Snap Rentals 已含 Simply Better Cover（含 CDW），富邦 3226CT20D09795 作為額外自負額補償
- **OKI**：Dream Rental Car 已含①免責補償保險+②NOC 補償，富邦 3226CT20D09792 作為額外補充
- 理賠時需提供租車公司的正式扣款明細
- **注意**：保障對象是你被收取的「自負額費用」，並非車輛全額損失

### 💀 意外身故 / 重大失能（最重要但最少用到）

- OKI 行程：富邦 300萬（#4）
- SIN/MEL/NZ 行程：富邦 500萬（#2）+ 兆豐（金額未確認）
- 搭乘飛機時發生意外，可能有大眾運輸加成（需查 PDF）

## 4. 理賠時可能需要準備的文件

| 理賠情境 | 需要的文件 | 備註 |
|----------|------------|------|
| **班機延誤** | 登機證、航空公司書面延誤聲明（蓋章，含原因）、相關費用收據 | 聲明書注意要有官方印章 |
| **班機取消** | 原訂機票、取消確認通知、重新購票收據（若有）| 需再向保險公司確認 |
| **行李延誤** | PIR 正本、登機證、行李標籤、緊急採購收據 | PIR 於下機後立即向航空公司申辦 |
| **行李遺失** | PIR 正本、航空公司遺失確認文件、購置清單（含估值）、收據 | 確認遺失通常需等 21 天 |
| **海外急診 / 就醫** | 醫療費用收據正本、診斷書、處方箋、醫院費用明細 | 正本，不接受影本或掃描件 |
| **租車損傷** | 租車契約、損傷確認報告、修車費 / 自負額扣款收據（官方文件）、警察事故報告（若有）| 需保留租車公司官方收據 |
| **意外身故 / 失能** | 醫療診斷書、警察報案、死亡證明（若適用）| 需再向保險公司確認完整清單 |

## 5. 不確定 / 看不清楚的地方

以下項目因保單 PDF 受密碼保護無法讀取，無法確認精確數字或條款：

1. 兆豐産險：**所有具體保額**（意外身故 / 醫療 / 各項不便險上限）
2. 兆豐産險：**班機延誤觸發時數**（幾小時後才賠？每幾小時賠一次？上限幾次？）
3. 富邦旅平險：**傷害醫療費用上限**（計畫二 vs 計畫三 分別多少）
4. 富邦旅平險：**海外突發疾病醫療費用上限**
5. 富邦旅平險：**班機延誤 / 行李延誤 / 遺失的具體保額**
6. 富邦租車附加：**每次事故補償上限**（TWD 或外幣）
7. 富邦租車附加：**NOC 費用是否涵蓋**
8. 兆豐 0800-053-588：**國際撥號方式**（境外如何撥打台灣 0800？）
9. **兩家保單班機延誤/行李保障是否可同時申請**（雙重投保問題）
10. **富邦旅平險被保人名單**（是否同時包含子擎和佩璇？）
